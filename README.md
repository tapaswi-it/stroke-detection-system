# Stroke Detection System

## Overview 
Brain Stroke Detection Based on CT Images Using Deep Learning

## Tech Stack
- **ML**: TensorFlow/Keras (CNN + LSTM)
- **Backend**: FastAPI
- **Frontend**: React + Bootstrap
- **Database**: PostgreSQL with Firebase Authentication
- **Containerization/Deployment**: Docker, Kubernetes
- **Version Control**: GitHub

## Workflow and Contribution Guidelines:
- **Main branch**: stable, always demo-ready.
- **Feature branches**: each dev works on a feature (frontend, backend, ML, integration).
- **Pull requests (PRs)**: must pass review before merging into main.
- **Commit messages**: structured, e.g., FE: Added file upload dashboard or ML: Implemented CNN training pipeline.
